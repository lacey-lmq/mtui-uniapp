## grids 参数
| 参数 | 默认值  | 类型   | 说明         | 可选值         |
| ---- | ------- | ------ | ------------ | -------------- |
| name | default | String | 九宫格的类型 | title，default |
| col  | 4       | Number | 一行的数量   | 2，3，4，5     |
| data | -       | Array  | 九宫格数据   | -               |


#### data说明
| 参数     | 默认值                                                                              | 类型   | 说明     | 可选值 |
| -------- | ----------------------------------------------------------------------------------- | ------ | -------- | ------ |
| icon     | ![icon_tabbar_3x.png](https://m.firstui.cn/static/images/common/icon_tabbar_3x.png) | String | 图标     | -      |
| menuName | 测试模块                                                                            | String | 标题     | -      |
| url      | -                                                                                   | String | 模块地址 | -       |
