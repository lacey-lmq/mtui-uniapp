## navbar参数
| 参数  | 默认值  | 类型    | 说明             | 可选值 |
| ----- | ------- | ------- | ---------------- | ------ |
| fixed | false   | Boolean | 是否固定在顶部   | -      |
| back  | true    | Boolean | 是否显示返回箭头 | -      |
| emit  | false   | Boolean | 是否自定义方法   | -      |
| title | -    | String  | navbar标题       | -      |
| nobg  | false   | Boolean | 是否无背景       | -      |
| bg    | #0762ed | String  | 背景色           | -       |
